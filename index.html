Rebound
Spring Animations for Android
About

Rebound is a simple java library that models spring dynamics. These Spring models can be used to create animations that feel natural by introducing real world physics to your application.

Demo

Press and release the Android icon to toggle a Spring from zero to one and observe the spring model drive a scaling animation.



Usage
{% highlight java %} // Create a system to solve the spring physics equation. SpringSystem springSystem = SpringSystem.create(); // Add a spring to the system. Spring spring = mSpringSystem.createSpring(); // Add a listener to observe the motion of the spring. spring.addListener(new SimpleSpringListener() { @Override public void onSpringUpdate(Spring spring) { // You can observe the updates in the spring state by asking its current // value in onSpringUpdate. float xTranslate = (float) spring.getCurrentValue(); myView.setTranslationX(xTranslate); } }); // Set the spring in motion; moving from 0 to 50. spring.setEndValue(50); {% endhighlight %}
